# Rendelések (Order endpoint)

A MiniCRM Order és Invoice moduljaiban található kártyák speciális felépítésűek (két füllel rendelkeznek). A rendelés-specifikus funkciók egy külön lapon és külön API végponton keresztül érhetők el.

Itt lehet listázni a rendeléshez hozzáadható tételeket, az egyedi rendelési mezők szerkesztését és a véglegesítést a dokumentáció további részei alapján.
